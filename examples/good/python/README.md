# Good Python API Example

This folder contains a straightforward FastAPI example showing:

- Clean router layer (`posts_router.py`)
- Service layer (`services.py`) with business logic
- In-memory repository (`repositories.py`) as a stand-in for a real database
- Pydantic DTOs and validation (`dto.py`)
- Structured error handling and consistent response shapes

This pattern mirrors the structure used in the mini projects (`projects/BlogApi/python`).
