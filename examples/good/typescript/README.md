# Good TypeScript API Example

This folder demonstrates a **clean, production-style** API layer using:

- Express routes (`postsRouter.ts`)
- Controller-style routing with error handling
- Service layer (`PostService.ts`) with business rules
- Repository layer (`PostRepository.ts`) for persistence (in-memory for the example)
- Clear validation and consistent error responses

## How to run

This example is designed to be read and understood as a standalone snippet. It does not run as a stand-alone server here, but is referenced from `projects/BlogApi/typescript` for a full runnable app.
