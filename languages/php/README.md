# PHP (Laravel-style patterns)

This folder contains language-level helpers and shared conventions for the PHP examples.

## Setup

Install dependencies from the root of the repo:

```bash
composer install
```

## Running an example

From the root, run:

```bash
php -S localhost:8001 -t projects/BlogApi/php bootstrap.php
```

Replace `BlogApi` with `EcommerceApi` to run the other mini project.
